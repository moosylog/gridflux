# ⚡ GRIDFLUX

A high-speed, Tetris-like typing game for modern ergonomic keyboards.
   
**GridFlux** is a dynamic typing trainer that turns learning a new keyboard into an immersive esports experience. Using falling "data nodes" and color-coded visual aids, it builds perfect muscle memory through reactive gameplay.

🌐 Overview

GridFlux is not just another typing test. It is a visual coordination tool built especially for people transitioning to professional and ergonomic split keyboards. 

Instead of typing rows of static text, GridFlux trains you to:
* **Identify Finger Zones:** Every key and falling node is color-coded to a specific finger.
* **React in 3D:** Watch the grid move and glow as you clear targets before they hit the ground.
* **Master Complex Layouts:** Practice everything from basic letters to advanced thumb-cluster navigation.

It is built as a single HTML file that runs directly in your browser. No installation, no setup tools, and no extra downloads—just open and play.

🚀 Key Features

**Color-Coded Learning:** The game uses a strict color system (Pinky, Ring, Middle, Index, Thumb) to provide a constant visual aid, ensuring you never use the wrong finger for a key.

**Tutor Hands:** Live on-screen hand overlays show you the exact resting position and movement required for every incoming node.

**Dynamic Game Modes:** Choose between **Training Mode** (infinite health and steady flow) for learning, or **Arcade Mode** (strict health and glitch effects) for high-stakes practice.

**Superpower System:** Build your "Power Bar" through accurate typing to trigger hyper-focus effects, shields, and massive score multipliers.

**Live Esports HUD:** Track your WPM (Words Per Minute), Combo streaks, and Level progress via a sleek, futuristic interface.

📁 Custom Keyboard Setup

Upload a JSON layout file in the setup menu. GridFlux will:
1. **Read your layout** configuration.
2. **Map your keys** correctly to the 3D grid.
3. **Adjust tutor hands** to match your physical hardware.
No manual coding or CSS overrides needed.

🕹️ How to Play

* **Match the Color:** Use the finger that matches the color of the falling node.
* **Clear the Grid:** Type the character on the node before it reaches the keyboard stage.
* **Maintain the Combo:** Avoid mistakes to trigger "Hyperdrive" visual effects and bonus points.
* **Activate Superpower:** Hit **Spacebar** when your bar is full to trigger an emergency shield or overcharge.

⚠️ Node Types

* **Standard:** Basic data nodes for steady rhythm.
* **Juggernaut:** Large, high-priority nodes that require focus.
* **Golden:** Rare nodes that provide massive score boosts.
* **Decoy:** Glitched red nodes—avoid typing these to stay alive.
* **Word Blocks:** Rapid-fire text strings that test your true speed.
