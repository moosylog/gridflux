## 🌐 Overview

GridFlux is not a regular typing test. It is a **visual coordination tool** built specifically for people transitioning to professional and ergonomic split keyboards.

Instead of typing rows of static text, GridFlux trains you to:
- **Identify Finger Zones:** Every falling node is color-coded to a specific finger.
- **React in 3D:** Watch the grid move and glow as you clear targets in real-time.
- **Master Muscle Memory:** Learn exactly where to place your fingers without looking down.

It is built as a single HTML file that runs directly in your browser. No installation, no setup tools, and no extra downloads—just open and play.

---

## 🚀 Key Features

* **Color-Coded Learning:** Uses a strict color system (Pinky, Ring, Middle, Index, Thumb) to provide a constant visual aid, ensuring perfect touch-typing technique.

* **On-Screen Tutor Hands:** Live hand overlays show you the exact resting position and movement required for every incoming key.

* **Dynamic Game Modes:** Switch between **Training Mode** (infinite health for beginners) and **Arcade Mode** (strict health and glitch effects for pros).

* **Superpower System:** Build your Power Bar through accuracy to trigger hyper-focus effects, shields, and massive score multipliers.

* **Live Esports HUD:** Track your live WPM (Words Per Minute), Combo streaks, and Level progress via a sleek, futuristic interface.

### 📁 Custom Keyboard Setup

Upload a JSON layout file in the setup menu. GridFlux will:
- Read your layout configuration  
- Map your keys correctly to the 3D grid  
- Adjust tutor hands to match your physical hardware  

No manual coding or CSS overrides needed.

---

## 🕹️ How to Play

1. **Match the Color:** Use the finger that matches the color of the falling node.  
2. **Clear the Grid:** Type the character on the node before it reaches the keyboard.  
3. **Keep the Combo:** Avoid mistakes to trigger "Hyperdrive" visual effects.  
4. **Trigger Superpower:** Hit **Spacebar** when your bar is full for an emergency shield.  

---

## ⚠️ Node Types

* **Standard:** Basic data nodes for steady rhythm.  
* **Juggernaut:** Large, high-priority nodes that require focus.  
* **Golden:** Rare nodes that provide massive score boosts.  
* **Decoy:** Glitched red nodes—avoid typing these to stay alive.  
* **Word Blocks:** Rapid-fire text strings that test your true speed.
